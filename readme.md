# lwe8-dts

## content

## Overview

Getting the DTS from a JavaScript file.

## Install

```bash
npm i lwe8-dts
```

```bash
pnpm i lwe8-dts
```

## Use

### cli

```bash
npx dts -f bar/foo.js #path to js file
```

### Common Js

```js
var { compile } = require("lwe8-dts");
```

### ESM

```js
import  { compile } from "lwe8-dts"
```
